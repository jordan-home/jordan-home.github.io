# 知道但做不到的根本原因

王阳明说"知行合一"，阿德勒说"决定我们行为的不是知识，是赋予它的意义"。

这两句话指向同一个问题：为什么我们学了很多道理，但还是做不到？

---

## 知识不是行动的原因

我们习惯性地认为：知道 → 应该做到。

但阿德勒的目的论颠覆了这个逻辑。

他说：决定我们行为的不是过去的经历，也不是我们"知道"什么，**而是我们为这个行为赋予了什么样的目的**。

"我懂时间管理，但我还是拖延"——目的不是管理时间，是通过拖延来避免失败的可能性。

"我知道运动好，但我不想动"——目的不是健康，是拖延本身在服务某个你不愿意承认的需求。

---

## 三个常见的"知道但做不到"

### 1. 读书

知道读书重要，但一年读不完两本。

阿德勒会问：读书"没有行动"，服务了什么目的？

可能的答案：读书是一件需要"认真对待"的事，如果我随便翻翻就开始行动，会显得不够"认真"。用"没读完"来维持"如果我认真读一定能行"的幻觉。

### 2. 副业

知道副业重要，但永远在"准备中"。

阿德勒会说：永远在准备的人，不是在追求成功，是在避免失败。

真正的失败需要被面对。而"还在准备"让这个面对永远不发生。

### 3. 健康

知道减脂要少吃多动，但体重纹丝不动。

目的不是"减不下来"，是维持现状在服务某种心理需求——也许是某种身份认同，也许是某种安全感。

---

## 怎么办？

阿德勒给过一个具体的操作：**不要问"我知道什么"，问"我赋予这个行为什么目的"**。

每次你说"我知道但做不到"的时候，停下来问自己：

- 我这个"做不到"，在服务什么目的？
- 如果我"做到了"，我会失去什么？
- 这个"失去"真的会发生吗，还是我脑补的？

看见目的，是改变的第一步。
